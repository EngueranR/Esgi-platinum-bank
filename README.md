# Platinum Bank

## Project setup

Installation des dépendances :

```bash
npm install
```

Configuration des variables d'environnement :

```bash
cp .env.exemple .env
```

Et modifier le fichier `.env` en l'ajustant à votre configuration.

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
